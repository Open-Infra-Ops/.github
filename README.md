# 开源基础设施运维团队

## 团队追求： 
构建最稳固、最可信、最好用的开源基础设施云底座

## 团队目标：
对开源基础设施SLA、成本、安全、架构负责；
特别是确保架构稳固：管控平台输入输出；尽可能多提供社区支持，最终实现运维平台自维护；
